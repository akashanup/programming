# Find All Subsets of a List of size k

Given a list, print all the sublist of a list of a given size k.

### Example 1
```sh
Input  : list = [1, 2, 3, 4, 5], k = 3 
Output : 
[
  [1, 2, 3]
  [1, 2, 4]
  [1, 2, 5]
  [1, 3, 4]
  [1, 3, 5]
  [1, 4, 5]
  [2, 3, 4]
  [2, 3, 5]
  [2, 4, 5]
  [3, 4, 5]
]
```
