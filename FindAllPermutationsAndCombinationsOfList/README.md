# Find All Permutations and Combinations of a List

Given a list, print all the permutations and combinations of a list.

### Example 1
```sh
Input  : list = [1, 2, 3] 
Output : 
  Combinations: [[], [1], [2], [3], [1,2], [1,3], [2,3], [1,2,3]] 
  Permutations: [[1,2,3], [1,3,2], [2,3,1], [1, 2, 3], [2], [2, 3], [3]]
```
