# Sum of Square Numbers

Given a non-negative integer c, decide whether there're two integers a and b such that a2 + b2 = c.

### Example 1
```sh
Input: c = 5
Output: true
Explanation: 1 * 1 + 2 * 2 = 5
```

### Example 2
```sh
Input: c = 3
Output: false
```

### Example 3
```sh
Input: c = 4
Output: true
```

### Example 4
```sh
Input: c = 2
Output: true
```

### Example 5
```sh
Input: c = 1
Output: true
```

### Constraints
```sh
0 <= c <= 2^31 - 1
```
