# Reverse Linked List

Given the head of a singly linked list, reverse the list, and return the reversed list.

[![rev1ex1](rev1ex1.jpg)]()
### Example 1
```sh
Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]
```

[![rev1ex2](rev1ex2.jpg)]()
### Example 2
```sh
Input: head = [1,2]
Output: [2,1]
```

### Example 3
```sh
Input: head = []
Output: []
```

### Constraints
```sh
The number of nodes in the list is the range [0, 5000].
-5000 <= Node.val <= 5000
```
